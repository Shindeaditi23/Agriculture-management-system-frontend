🌾 Agriculture Management System (Frontend)
📌 Project Description

The Agriculture Management System is a frontend web project designed to provide information about modern farming and agricultural practices. It includes multiple pages such as Home, About, Departments, Facilities, and Contact, all connected through a navigation menu. The project is built using HTML and CSS to create a clean, attractive, and user-friendly interface.

🎯 Objectives
To design a multi-page website using HTML
To apply CSS for styling and layout
To create proper navigation between pages
To display agriculture-related information effectively
🛠️ Technologies Used
HTML5
CSS3
